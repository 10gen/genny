🧞‍🐍 Genny Python  🧞‍🐍
========================

Quick-Start
-----------

```sh
pip install virtualenv
virtualenv venv
source ./venv/bin/activate
pip install .
```

Also consider using [pyenv](https://github.com/pyenv/pyenv) to ensure you're using the correct version of python.

Self-Tests
----------

```sh
pip install virtualenv
virtualenv venv
source ./venv/bin/activate
pip install .
python setup.py test
```
